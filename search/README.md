# AMFV Search

Takes a single medical claim and returns a handful of sources that directly
**support** or **contradict** it, drawn from a curated medical corpus. The
[verifier](../verifier/README.md) dispatches it whenever a claim is not in the
fact cache. Part of the [Agentic Medical Fact Verifier](../README.md).

Workspace member (`amfv-search`). Training experiments live in
[`../training`](../training/README.md).
