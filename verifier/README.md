# AMFV Verifier

Scores each atomic claim from the [decomposer](../decomposer/README.md) on a
five-level scale — **strongly supported**, **weakly supported**, **unclear**,
**weakly unsubstantiated**, **strongly unsubstantiated** — part of the
[Agentic Medical Fact Verifier](../README.md).

Workspace member (`amfv-verifier`). Training experiments live in
[`../training`](../training/README.md).
